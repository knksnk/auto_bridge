# AutoBridge React Frontend

Профессиональный React-фронтенд AutoBridge на базе дизайнерской верстки.

## Команды

```bash
npm install
npm run dev
npm run build
npm run preview
npm run lint
```

## Стек

- Vite
- React
- TypeScript
- React Router

Данные пока моковые и вынесены в `src/services`, чтобы позже заменить их на реальные backend API.
